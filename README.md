Book Library - Local Collection Manager
CPSC 3750 - Solo Project 2

Netlify url: https://resplendent-taiyaki-c6f46f.netlify.app/

Backend is written in php

Json Persistence: All data is stored in a server-side books.json file. The PHP backend handles all CRUD operations by reading from and writing to this file, ensuring data persists across refreshes and sessions. The frontend does not store data locally.
