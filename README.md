Book Library — Local Collection Manager
CPSC 3750 — Solo Project 1

This is a local Book Library web application that allows users to create, view, edit, and delete books. The application runs locally using XAMPP with Apache. All data is stored in JavaScript and persisted using localStorage so it remains after page refreshes.

Domain: Book Library  
Fields include title, author, genre, pages, publication year, and read/unread status.

How to run:
1. Start XAMPP and enable Apache.
2. Place the project folder in the htdocs directory with the folder name: book-library
3. Open a browser and go to http://localhost/book-library
